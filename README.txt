== Description ==

Add your Advent Calendar (from adventcalendar.com) to any page or post in Wordpress. We do not recommend displaying it in a widget area unless the width of the area is at least equal to an iPhone screen.

This plugin is made to support the Advent Calendar App in Wordpress and is **NOT** a free service.

Use the [my_calendar] shortcode to display calendar in any page/post.

**Available options**

* width (in px or %)
* height (in px)
* border (in px)
* bordercolor (in hex where ff0000 would be bright red)

See FAQ for further description and examples.

== Installation ==

1. Upload My-Christmas-Calendar to the `/wp-content/plugins/` directory or simply download it from Wordpress.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the Settings menu and edit My-Christmas-Calendar Settings
1. Type in your web address
1. Paste the [my_calendar] shortcode into any page/post (see FAQ for more options)

== Frequently Asked Questions ==
= Exampels & options =

* Basic: [my_calendar]
* With opitional attributes [my_calendar width="100%" height="800px" border="3" bordercolor="000000"] - notation is optional.

= Troubleshooting =
* If the calendar is broken you hav probably used the wrong domain. Check your Advent Calendar admin to view what you have selected.
